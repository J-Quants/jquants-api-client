# jquants-api-client


* ラッパー群　J-Quants API の各APIエンドポイントに対応しています。
  * get_idtoken
  * get_listed_info
  * get_listed_sections
  * get_market_segments
  * get_prices_daily_quotes
  * get_fins_statements
  * get_fins_announcement
* ユーティリティー群　日付範囲を指定して一括でデータ取得して、取得したデータを結合して返すようなユーティリティーメソッドです。
  * get_list
  * get_price_all
  * get_statements_all
